School Projects

  • Master\'s Project (Biomedical Image Segmentation Utilizing Hierarchical Spanning Trees with WebCL)
    I developed a web-based GUI using JavaScript that allows a user to upload an image. The image will then be sent to a parallel computing device on their computer using WebCL (an implementation of OpenCL for use in web browsers). This process will then take the image a split it up into a hierarchy of clusters. The user will then be able to select multiple points on the image and export a new image containing the smallest cluster that contained all selected points. The idea behind the research is to aid in cancer detection by automatically detecting a change in coloration (and eventually match with a heatmap image of the same area) and compare the border of the potential cancerous cells with that of known cancer.

  • Image Segmentation with CUDA
    Utilizing the passive parallelism of CUDA, I created a C++ program that could take an image as input and generate a hierarchical spanning tree of the clusters found in that image, judging clusters as the distance from pixel to pixel as well as the color difference between each pixel.   • Protein Structure Representation
    Given a set of proteins along with their distances from each other, we created a program that will take the proteins and create a graphical representation of the structure of the molecule in 3D space. This worked by performing a function similar to that of triangulation in 3D space. The first point could be anywhere in 3D space, so we assign that as the origin (0, 0, 0). The second point could be anywhere on a sphere around the first point, so we assign that along the x-axis. The third point will be anywhere on a ring (intersection of two spheres of the first two points), so we assign that on the positive y-axis. The fourth point will be at one of two locations, where all three spheres of the previous three points intersect, we select the positive z-axis. Using those four points, we have defined our own 3D coordinate space and we\'re able to plot all of the remaining points.
  • Surgical Simulation
    Utilizing nVidia PhysX and Novint Falcon, we created a program that could simulate performing surgery on a small section of skin, allowing the user to test surgery techniques in a simulated environment. We achieved this by allowing a cutting tool to be controlled in 3D space and creating a soft body in PhysX that could be destroyed by the tool.
  • Heat Based Cancer Detection
    As a Microsoft Imagine Cup project, I worked on a team to develop a tool that utilized an Arduino to connect to a Windows 8 Phone. The Windows 8 Phone would allow the user to control the Ardino via bluetooth, the Arduino would communicate with an infrared sensor to scan a section of skin and transimate the sans back to the Windows 8 Phone, which would then be shown to the user and sent to the Azure web server to be processed to help determine if a given scanned area is cancerous.
